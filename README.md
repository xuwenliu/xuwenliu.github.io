### 1.如何新建文章
在source/_posts目录下新建md文件

### 2.如何开启本地预览
执行命令 hexo s 或者 yarn server

### 3.如何上传并自动部署到github
依次执行命令 
+ hexo clean    yarn clean
+ hexo g        yarn build
+ hexo d        yarn deploy

### 4.如何上传源码
+ cd 进入xuwenliu.github.io目录 且git对应分支为source
+ git add .
+ git commit -am ""
+ git push
